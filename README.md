# IPL Data Analysis and Visualization
## This project analyzes the Indian Premium League (IPL) matches data to uncover insights such as:
- Most Win team and Loss team overall season
- Highest Win by Runs and Wickets
- Toss Decision and Match Outcomes
- Batsman who have most run overall season
- Baller who have most wicket overall season
- Orange Cap Holder of each season
- Purple Cap holder of each season

## Methodology:
* Collected IPL Matches Data from kaggle.
* Cleaned and processed data using Pandas.
* Visualized result using Seaborn and Matplotlib.
* Add bullet points for insights below each visualization to highlight the key finding.
